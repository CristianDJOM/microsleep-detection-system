## 📖 Project Description

Microsleep Detection System is a research project designed to improve road safety by detecting microsleep episodes in drivers in real time. The system integrates computer vision techniques with embedded hardware to monitor the driver's facial behavior and identify signs of drowsiness that may lead to accidents.

The solution is composed of two main components. The first is a Raspberry Pi equipped with a night vision camera, responsible for continuously monitoring the driver and detecting microsleep events. When a microsleep episode is identified, the system activates an audible alarm through a speaker to alert the driver and help prevent potential accidents.

The second component is a web platform developed with Node.js, which allows the registration of drivers, storage of detected events, and visualization of microsleep statistics. This provides valuable information for monitoring driver behavior and analyzing fatigue-related incidents.

The project was presented at regional and international research events organized by RedCOLSI, demonstrating the integration of software development, embedded systems, computer vision, and road safety technologies.

## 🎯 Objective

Develop an intelligent monitoring system capable of detecting microsleep episodes in drivers, generating real-time alerts, and recording events for subsequent analysis in order to reduce the risk of traffic accidents caused by fatigue.

## ✨ Features

- Driver registration and management.
- Real-time microsleep detection.
- Audible alerts for driver awareness.
- Night vision monitoring.
- Event logging and statistics visualization.
- Web-based management platform.
- Integration between hardware and software components.
